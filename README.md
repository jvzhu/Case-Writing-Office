# Case Writing Office Repository

## Overview
This repository contains case studies, teaching cases, research materials, and documentation for the Case Writing Office at Stanford Graduate School of Business (cwo@gsb.stanford.edu).

## Project Goals
- Collect and organize teaching case materials
- Document case development processes
- Facilitate case writing and publication workflows

## Getting Started

### Prerequisites
- Git
- Basic familiarity with GitHub workflows

### Setup
1. Clone the repository
2. Review the issues and project board for active tasks
3. Refer to the CONTRIBUTING.md for guidelines
